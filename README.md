# 🛒 E-Commerce Website

A fully functional front-end eCommerce website built using HTML, CSS, and JavaScript.  
This project allows users to browse products, add items to the cart, and complete checkout.

🔗 **Live Website:**  
https://JabinSultana.github.io/ecommerce-website/

🔗 **GitHub Repository:**  
https://github.com/JabinSultana/ecommerce-website

---

## 🚀 Features

- 🛍️ Product listing with images and prices
- 🛒 Add to Cart functionality
- ➕ Increase/Decrease product quantity
- 💳 Checkout page
- ✅ Order success page
- 📱 Responsive design
- ⚡ Dynamic product loading using JavaScript

---

## 🧰 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Git & GitHub
- GitHub Pages (for deployment)

This project is deployed using GitHub Pages.

To run locally:

1. Download or clone the repository
2. Open `index.html` in your browser
